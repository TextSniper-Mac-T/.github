## Overview

TextSniper Advanced is an OCR text capture application that instantly recognizes and extracts text from anywhere on screen — images, videos, presentations, PDFs, screenshots, non-selectable interface elements, and any other visual content that contains text but does not allow selection. A single keyboard shortcut activates a screen capture crosshair, you draw a rectangle around the text, and the recognized content appears on your clipboard ready to paste. The machine learning recognition engine handles printed text, handwritten text, and text in images with high accuracy.

Screen OCR captures text from any visual content on screen by selecting a region, recognizing text within that region, and placing the result on the clipboard. The recognition works across any application, any window, and any content type that displays text visually. This capability eliminates the tedious process of manually retyping text that appears in images, videos, or non-selectable interface elements.

Machine learning recognition uses trained neural networks running on the Neural Engine for fast, accurate text recognition that handles multiple languages, various fonts, and challenging conditions including low contrast, curved text, and partially obscured characters. The recognition processes locally without sending screen content to external servers.

QR code and barcode reading decodes QR codes and barcodes visible on screen, extracting the encoded data — URLs, contact information, product codes — without requiring a separate scanner application. The same selection gesture that captures text also detects and decodes machine-readable codes.

For anyone who encounters text in images, videos, or non-selectable content and needs to extract it without retyping, TextSniper Advanced is the OCR tool that makes text extraction a single keyboard shortcut away from any screen content.

## Performance

TextSniper balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
